# Wisdom's Creatives Website

This repository contains the business website for Wisdom's Creatives.

## How to Upload to GitHub Pages

1. Go to GitHub and create a **new repository** named `wisdomcreatives.github.io`.
2. Upload the contents of this ZIP file (`index.html` and `logo.png`) to the repository.
3. Make sure the main branch is selected and the files are in the root directory (not in a folder).
4. After uploading, go to **Settings â†’ Pages** and select the main branch as the source (root folder).
5. Your website will be live at: `https://wisdomcreatives.github.io`

You can replace `logo.png` with your actual logo anytime.
